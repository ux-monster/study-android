# Notification

NotificationManager 클래스를 사용하여, 푸시알림을 나타내기 위한 정보를 담고 있는 클래스입니다.

## Notification.Action

이 알림의 일부로 표시할 수 있는 명명된 액션을 캡슐화하는 구조체입니다. 여기에는 아이콘, 레이블, 사용자가 액션을 선택할 때 실행될 PendingIntent가 포함되어야 합니다.

- Notification.Action.WearableExtender
